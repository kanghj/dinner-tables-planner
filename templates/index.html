Creating the Guest List

Upload an Excel file listing each group/community of guests. Write the name of the community as the header of each column. Then add names of guests in that community under that column.

A single person is allowed to appear in multiple columns. E.g. In this wedding, "Hercule Poirot" is a classmate to both the groom and the bride, and appears in both columns.
